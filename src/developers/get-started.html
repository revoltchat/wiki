Revolt Developer Introduction

Welcome! This page is for developers who want to build on Revolt—an open-source chat platform designed for customization and creative freedom.

What You Can Build

  • Bots and automation tools
  • Custom clients and extensions
  • Themes and visual enhancements
  • Core contributions to the platform

Getting Started with the API

Revolt offers a REST API and WebSocket gateway for real-time events. You can:

  • Send and receive messages
  • Listen for events like messageCreate, userUpdate, etc.
  • Manage servers, channels, roles, and users

API Docs: https://publicapi.dev/revolt-api

Use a Library

To simplify development, use a community SDK:

  • JavaScript: revolt.js – Guide
  • Python: Revolt.py
  • C#: Revolt.NET
  • Go: RevoltGo

Quick Example

client.on("messageCreate", (message) => {
  if (message.content === "!ping") {
    message.reply("Pong!");
  }
});
